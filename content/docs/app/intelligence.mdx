The Intelligence Page is your AI-powered project briefing room. It aggregates completed work into a timeline and surfaces proactive insights in the Briefings tab.

## Overview

The Intelligence Page consists of two main sections accessed via tabs:

- **Timeline**: A chronological feed of recent work, sorted newest to oldest
- **Briefings**: AI-generated opportunities and suggestions for improvements

## Timeline Tab

The Timeline shows a reverse-chronological list of completed activities. Each item displays:

- **Primary text**: What was completed (e.g., "Fixed Navigation Bug")
- **Timestamp**: When the work was completed (e.g., "2 mins ago")
- **Icon**: Visual indicator of the activity type

- Context (which chat or investigation it came from)

## Taking Action

Both Timeline and Briefing detail sheets offer two primary actions:

### Open in Chat
Creates a new Chat tab pre-populated with the context from the selected item. This allows you to:

- Ask follow-up questions
- Request modifications
- Explore related code

### Start Investigation
Triggers the AI agent to perform a deeper analysis of the item. This opens [The Desk](./desk) where you can monitor the investigation in real-time.

<Callout type="tip">
Use "Open in Chat" for conversational exploration. Use "Start Investigation" when you want the agent to autonomously analyze and report findings.
</Callout>

## Empty States

When there's no data to display, the Intelligence Page shows helpful empty states:

- **Timeline**: "No activity yet. Open a tab to get started."
- **Briefings**: "All clear. Shadow Agent will notify you of opportunities."
