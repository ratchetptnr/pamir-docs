## Before You Start

**What You Need:**
- Your Distiller (unboxed and plugged in)
- Your smartphone (iOS or Android)
- WiFi network name and password
- 5 minutes of time

**Important Notes:**
- Your Distiller supports **2.4GHz and 5GHz WiFi**
- Keep your phone near the device during setup
- Bluetooth must be enabled on your phone

---

## What Happens After Setup?

**Device Display:**
- Shows dashboard with system status
- Displays IP address
- No longer shows QR code

**App:**
- Shows your device as "Online"
- Green status indicator
- You can now access all features

**Bluetooth:**
- Automatically disconnects (no longer needed)
- Turns OFF to save battery
- Will turn back ON if WiFi is lost

## Common Questions

<Accordions>

<Accordion title="Can I change the device name later?">

Yes! Go to Settings in the app â†’ Device Info â†’ Edit Name

</Accordion>

<Accordion title="What if I forget the device password?">

You'll need to factory reset the device. This will erase all settings and you'll need to set it up again from scratch.

To factory reset:
1. Hold the power button for 10 seconds
2. Device will show "Factory Reset?"
3. Confirm by pressing the button again

</Accordion>

<Accordion title="Can I connect to a different WiFi network later?">

Yes! Open the app â†’ Settings â†’ Change WiFi Network

You'll need to enter your device password to authorize the change.

</Accordion>

<Accordion title="Why does it take so long to connect to WiFi?">

The device is doing several things:
1. Joining the WiFi network (5-10 seconds)
2. Getting an IP address from your router (5 seconds)
3. Verifying internet connectivity (5 seconds)
4. Establishing secure connection to Pamir servers (5-10 seconds)

Total: 20-30 seconds is normal.

</Accordion>

<Accordion title="What happens if I quit the app during setup?">

**Good news:** You can continue where you left off!

- If you quit before setting the password: Start from the beginning
- If you quit after setting the password:
  1. Reopen the app
  2. App will find your device via Bluetooth
  3. Enter your device password
  4. Continue from WiFi setup

</Accordion>
