## Quick Reference

### Display
- Adjust contrast and refresh rates
- Configure dark mode behavior
- Set display timeout preferences

### Audio
- Control volume levels
- Configure notification sounds
- Enable/disable feedback sounds

### LED
- Understand status indicator meanings
- Configure LED brightness
- Enable/disable LED notifications

### Storage
- Monitor available space
- Clear cached data
- Manage project files

### Updates
- Check for software updates
- Configure automatic updates
- View update history
