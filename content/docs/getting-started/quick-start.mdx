### Step 4: Set Device Password

Create a password to protect your device. You'll need this password if you ever need to recover or change settings.

**What to do:**
1. Create a strong password (minimum 6 characters, but 8+ recommended)
2. Confirm the password
3. Tap Save

**Why both name and password?**
- **Name** = Device identification (which physical device is this?)
- **Password** = Device authorization (who can change settings?)

This is like naming and locking your iPhone.

You can now:

- Start creating projects and agents

### If WiFi Is Lost Later

If your Distiller loses internet connection:
1. The device will show the same setup screen with a QR code
2. Scan the QR code and repeat the WiFi setup process
3. It's the same simple process as the initial setup
